# Todoz Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `todoz`, a standalone Go CLI binary that manages todo lists/tasks via an append-only event log, exposing a language-agnostic JSON-over-subprocess interface for any consuming app.

**Architecture:** Single Go binary. Apps invoke `todoz <command> [flags]` as a subprocess and read a JSON envelope from stdout. State is an append-only `events.jsonl` log; every command loads the full log, computes state, applies a mutation, atomically appends a new event, and exits. No daemon, no background process. Soft-delete trash (nothing is ever truly removed). Every request is recorded to a per-app operation log with microsecond Istanbul-timezone timestamps.

**Tech Stack:** Go (stdlib only — no third-party deps), `encoding/json`, `os`, `time`, `crypto/rand`, file locking via `O_CREATE|O_EXCL` lock file. Tests use Go's built-in `testing` package.

**Implementation Principles (from spec §11):** English only. KISS. YAGNI. TDD (red→green→refactor). Hyper-documented (doc comment on every exported symbol). Extreme modularity (small single-purpose files, domain folders). No file should hold more than one responsibility.

**Decomposition rationale:** `model` holds pure data + pure functions (no I/O). `events` defines the log vocabulary and the reducer. `state` is the computed view. `store` owns all disk I/O and locking. `oplog` is the audit trail (separate concern from the event log). `response` is the output contract. `cli` wires flags → command → store → response. `cmd/todoz` is a thin shell. Every layer is independently testable.

## Conventions Used Throughout

- **Module path:** `todoz` (so imports are `todoz/internal/...`).
- **Timestamps:** RFC3339 with microseconds and fixed `+03:00` offset, e.g. `2026-06-03T10:30:45.123456+03:00`. Produced only via `clock`.
- **Dates:** `YYYY-MM-DD` strings, no time component.
- **Errors to the caller:** never `panic`; always a JSON error envelope + non-zero exit code.
- **Doc comments:** every exported type/func begins with a `// Name ...` comment (Go convention), per the hyper-documented requirement.

// Package clock is the single source of time for todoz.
//
// Every timestamp in the system (events, operation log, responses) is produced
// here so the format is uniform: RFC3339 with microsecond precision and a fixed
// +03:00 (Istanbul) offset. Istanbul has observed UTC+3 with no daylight saving
// since 2016, so a fixed zone is correct and requires no network/TZ database.
package clock

import "time"

// layout is RFC3339 with 6 fractional digits and a numeric offset.
const layout = "2006-01-02T15:04:05.000000-07:00"

// istanbul is the fixed +03:00 zone used for all timestamps.
var istanbul = time.FixedZone("+03:00", 3*60*60)

// Zone returns the fixed Istanbul (+03:00) location.
func Zone() *time.Location { return istanbul }

// Now returns the current wall-clock time expressed in the Istanbul zone.
func Now() time.Time { return time.Now().In(istanbul) }

// Format renders t using the canonical todoz timestamp layout.
func Format(t time.Time) string { return t.In(istanbul).Format(layout) }
```

// Package ids generates unique identifiers for tasks and lists.
//
// IDs are of the form "<prefix>-<32 hex chars>" using 16 cryptographically
// random bytes. Random IDs (rather than sequential counters) keep generation
// lock-free and collision-free across concurrent todoz processes.
package ids

import (
	"crypto/rand"
	"encoding/hex"
)

// New returns a unique id with the given semantic prefix (e.g. "task", "list").
func New(prefix string) string {
	var b [16]byte
	// crypto/rand.Read never returns a short read on supported platforms;
	// any error here is catastrophic (no entropy) and worth surfacing.
	if _, err := rand.Read(b[:]); err != nil {
		panic("ids: cannot read random bytes: " + err.Error())
	}
	return prefix + "-" + hex.EncodeToString(b[:])
}
```

// Package config resolves runtime configuration from environment variables
// and derives every filesystem path todoz uses.
//
// Environment variables:
//
//	TODO_LIB_HOME  Root directory for all todoz data. Default: <user config>/todoz.
//	TODO_APP_NAME  Identifier of the calling app, used in the operation log.
//	               Default: "unknown".
//
// Keeping all path construction in one place means the rest of the codebase
// never hard-codes a filename.
package config

import (
	"os"
	"path/filepath"
)

// Config is the resolved configuration for a single todoz invocation.
type Config struct {
	// Home is the root directory holding the event log, backups, and logs.
	Home string
	// AppName identifies the calling application in the operation log.
	AppName string
}

// Load reads configuration from the environment, applying defaults.
func Load() (Config, error) {
	home := os.Getenv("TODO_LIB_HOME")
	if home == "" {
		base, err := os.UserConfigDir()
		if err != nil {
			return Config{}, err
		}
		home = filepath.Join(base, "todoz")
	}

	app := os.Getenv("TODO_APP_NAME")
	if app == "" {
		app = "unknown"
	}

	return Config{Home: home, AppName: app}, nil
}

// EventsPath is the append-only event log file.
func (c Config) EventsPath() string { return filepath.Join(c.Home, "events.jsonl") }

// BackupPath is the pre-compaction backup of the event log.
func (c Config) BackupPath() string { return filepath.Join(c.Home, "events.jsonl.backup") }

// LockPath is the exclusive lock file guarding event-log appends.
func (c Config) LockPath() string { return filepath.Join(c.Home, ".lock") }

// LogsDir is the directory holding per-app operation logs.
func (c Config) LogsDir() string { return filepath.Join(c.Home, "logs") }

// AppLogPath is the operation log for the current app.
func (c Config) AppLogPath() string { return filepath.Join(c.LogsDir(), c.AppName+".log") }
```

package model

// ComputeOverdue reports whether a task should display as overdue relative to
// the given today date (YYYY-MM-DD). A task is overdue when it is still pending

import "encoding/json"

// Encode serializes an event to a single compact JSON string WITHOUT a trailing
// newline. The store's log layer is responsible for adding the newline so that
// events.jsonl remains one-event-per-line.
func Encode(e Event) (string, error) {

func Decode(line string) (Event, error) {

	}
	return e, nil

package store

import (
	"os"
	"time"
)

// Lock represents a held exclusive lock backed by a lock file created with
// O_CREATE|O_EXCL. Cross-process mutual exclusion relies on the OS guaranteeing
// that only one process can exclusively create a given path.
type Lock struct {
	path string
}

// Acquire obtains the lock at path, retrying until timeoutMS milliseconds
// elapse. It returns an error if the lock cannot be obtained in time, which
// prevents two todoz processes from appending to the log simultaneously.
func Acquire(path string, timeoutMS int) (*Lock, error) {
	deadline := time.Now().Add(time.Duration(timeoutMS) * time.Millisecond)
	for {
		f, err := os.OpenFile(path, os.O_CREATE|os.O_EXCL|os.O_WRONLY, 0o600)
		if err == nil {
			_ = f.Close()
			return &Lock{path: path}, nil
		}
		if !os.IsExist(err) {
			return nil, err // unexpected error (e.g. bad directory)
		}
		if time.Now().After(deadline) {
			return nil, err // timed out waiting for the holder to release
		}
		time.Sleep(2 * time.Millisecond)
	}
}

// Release removes the lock file, allowing another process to acquire it.
func (l *Lock) Release() error {
	return os.Remove(l.path)
}

package store

import (
	"bufio"
	"os"
	"path/filepath"
	"strings"

	"todoz/internal/events"
)

// ReadAll reads and decodes every event from the log at path, in order. A
// missing file is treated as an empty log (the normal first-run case). Blank
// lines are skipped defensively.
func ReadAll(path string) ([]events.Event, error) {
	f, err := os.Open(path)
	if err != nil {
		if os.IsNotExist(err) {
			return []events.Event{}, nil
		}
		return nil, err
	}
	defer f.Close()

	var out []events.Event
	sc := bufio.NewScanner(f)
	// Allow long lines (large descriptions) up to 1 MiB per event.
	sc.Buffer(make([]byte, 0, 64*1024), 1024*1024)
	for sc.Scan() {
		line := strings.TrimSpace(sc.Text())
		if line == "" {
			continue
		}
		e, err := events.Decode(line)
		if err != nil {
			return nil, err
		}
		out = append(out, e)
	}
	if err := sc.Err(); err != nil {
		return nil, err
	}
	return out, nil
}

// AppendEvent appends a single event as one line to the log at path, creating
// the parent directory and file if necessary. The write is O_APPEND so it is
// atomic for the small line sizes involved; callers still hold the store lock
// to serialize concurrent writers.
func AppendEvent(path string, e events.Event) error {
	if err := os.MkdirAll(filepath.Dir(path), 0o755); err != nil {
		return err
	}
	line, err := events.Encode(e)
	if err != nil {
		return err
	}
	f, err := os.OpenFile(path, os.O_CREATE|os.O_APPEND|os.O_WRONLY, 0o644)
	if err != nil {
		return err
	}
	defer f.Close()
	if _, err := f.WriteString(line + "\n"); err != nil {
		return err
	}
	return f.Sync()
}

package store

import (
	"todoz/internal/config"
	"todoz/internal/events"
	"todoz/internal/state"
)

// lockTimeoutMS bounds how long a process waits for the append lock before
// giving up. Five seconds is generous given that each held section is a few
// milliseconds; exceeding it indicates a stuck or crashed holder.
const lockTimeoutMS = 5000

// Store is the disk-backed entry point for reading and mutating todoz data.
// It owns the event log file and the append lock; higher layers never touch
// files directly.
type Store struct {
	cfg config.Config
}

// New returns a Store bound to the given configuration.
func New(c config.Config) *Store {
	return &Store{cfg: c}
}

// Load reads the entire event log and replays it into a fresh State. Full replay
// on every call is intentional: it guarantees each invocation sees the latest
// committed state with zero risk of stale reads.
func (s *Store) Load() (state.State, error) {
	evs, err := ReadAll(s.cfg.EventsPath())
	if err != nil {
		return state.State{}, err
	}
	st := state.New()
	for _, e := range evs {
		events.Apply(&st, e)
	}
	return st, nil
}

// Append commits one event to the log under the exclusive lock, guaranteeing
// that concurrent todoz processes serialize their writes and never interleave.
func (s *Store) Append(e events.Event) error {
	lock, err := Acquire(s.cfg.LockPath(), lockTimeoutMS)
	if err != nil {
		return err
	}
	defer lock.Release()
	return AppendEvent(s.cfg.EventsPath(), e)
}

package store

import (
	"os"

	"todoz/internal/events"
	"todoz/internal/model"
	"todoz/internal/state"
)

// Compact rewrites the event log so it contains the minimum number of events
// needed to reproduce the current state — one synthetic creation event per
// surviving list and task. The previous log is preserved as events.jsonl.backup.
//
// Compaction runs under the append lock so no writer can race it. The rewrite
// is done to a temp file and atomically renamed into place, so a crash mid-way
// never corrupts the live log.
func (s *Store) Compact() error {
	lock, err := Acquire(s.cfg.LockPath(), lockTimeoutMS)
	if err != nil {
		return err
	}
	defer lock.Release()

	evs, err := ReadAll(s.cfg.EventsPath())
	if err != nil {
		return err
	}
	st := state.New()
	for _, e := range evs {
		events.Apply(&st, e)
	}

	snapshot := snapshotEvents(st)

	tmp := s.cfg.EventsPath() + ".tmp"
	if err := writeEvents(tmp, snapshot); err != nil {
		return err
	}

	// Keep a backup of the old log, then swap the compacted log in atomically.
	_ = os.Rename(s.cfg.EventsPath(), s.cfg.BackupPath())
	return os.Rename(tmp, s.cfg.EventsPath())
}

// snapshotEvents produces the minimal event list that reconstructs st. Lists
// come first so their tasks can reference them on replay. Soft-delete and
// completion flags are re-expressed as follow-up events.
func snapshotEvents(st state.State) []events.Event {
	var out []events.Event
	for _, l := range st.Lists {
		out = append(out, events.Event{Type: events.TypeListCreated, At: l.CreatedAt, ListID: l.ID, ListName: l.Name})
		if l.IsDeleted {
			out = append(out, events.Event{Type: events.TypeListDeleted, At: l.CreatedAt, ListID: l.ID})
		}
	}
	for _, tk := range st.Tasks {
		out = append(out, events.Event{
			Type: events.TypeTaskCreated, At: tk.CreatedAt,
			TaskID: tk.ID, ListID: tk.ListID, Title: tk.Title, Description: tk.Description, Date: tk.Date,
		})
		if tk.Status == model.StatusCompleted {
			out = append(out, events.Event{Type: events.TypeTaskCompleted, At: tk.CompletedAt, TaskID: tk.ID})
		}
		if tk.IsHiddenTrash {
			out = append(out, events.Event{Type: events.TypeTaskPermanentlyDeleted, At: tk.UpdatedAt, TaskID: tk.ID})
		} else if tk.IsDeleted {
			out = append(out, events.Event{Type: events.TypeTaskDeleted, At: tk.DeletedAt, TaskID: tk.ID})
		}
	}
	return out
}

// writeEvents writes events to path, one JSON object per line.
func writeEvents(path string, evs []events.Event) error {
	f, err := os.OpenFile(path, os.O_CREATE|os.O_TRUNC|os.O_WRONLY, 0o644)
	if err != nil {
		return err
	}
	defer f.Close()
	for _, e := range evs {
		line, err := events.Encode(e)
		if err != nil {
			return err
		}
		if _, err := f.WriteString(line + "\n"); err != nil {
			return err
		}
	}
	return f.Sync()
}

// Package oplog writes the human-readable operation audit log. EVERY todoz
// invocation — success or failure — appends one block here, with microsecond
// Istanbul timestamps, the exact request, and the full response. This is
// separate from the event log: the event log is the data; the oplog is the
// chronological record of who asked for what and what happened.
package oplog

import (
	"fmt"
	"os"
	"path/filepath"
	"strings"
)

// Entry is one recorded request/response pair.
type Entry struct {
	// Timestamp is the canonical todoz timestamp the request was handled.
	Timestamp string
	// Request is the reconstructed command line (e.g. `todoz add-task ...`).
	Request string
	// OK indicates whether the operation succeeded.
	OK bool
	// Data is the JSON payload returned on success (empty on failure).
	Data string
	// Error/Message describe a failure (empty on success).
	Error   string
	Message string
	// DurationMS is the wall-clock handling time in milliseconds.
	DurationMS int64
}

// Record appends one block describing the entry to the log at path, creating
// the parent directory if needed. Logging failures are returned but callers
// generally treat them as non-fatal (the primary operation already happened).
func Record(path string, e Entry) error {
	if err := os.MkdirAll(filepath.Dir(path), 0o755); err != nil {
		return err
	}
	var b strings.Builder
	fmt.Fprintf(&b, "[%s]\n", e.Timestamp)
	fmt.Fprintf(&b, "REQUEST: %s\n", e.Request)
	fmt.Fprintf(&b, "RESPONSE: ok=%t\n", e.OK)
	if e.OK {
		fmt.Fprintf(&b, "DATA: %s\n", e.Data)
	} else {
		fmt.Fprintf(&b, "ERROR: %s\n", e.Error)
		fmt.Fprintf(&b, "MESSAGE: %s\n", e.Message)
	}
	fmt.Fprintf(&b, "DURATION: %dms\n\n", e.DurationMS)

	f, err := os.OpenFile(path, os.O_CREATE|os.O_APPEND|os.O_WRONLY, 0o644)
	if err != nil {
		return err
	}
	defer f.Close()
	_, err = f.WriteString(b.String())
	return err
}

// Package response defines the single JSON envelope every todoz command prints
// to stdout. A stable envelope is the contract consuming apps depend on: check
// "ok", then read "data" or the "error"/"message" pair.
package response

import "encoding/json"

// Envelope is the top-level JSON object returned by every command.
type Envelope struct {
	// OK indicates success.
	OK bool `json:"ok"`
	// Data is the success payload (omitted on failure).
	Data any `json:"data,omitempty"`
	// ErrCode is a stable machine-readable error code (omitted on success).
	ErrCode string `json:"error,omitempty"`
	// Message is a human-readable error description (omitted on success).
	Message string `json:"message,omitempty"`
}

// Success builds a successful envelope wrapping data.
func Success(data any) Envelope {
	return Envelope{OK: true, Data: data}
}

// Error builds a failure envelope with a code and message.
func Error(code, message string) Envelope {
	return Envelope{OK: false, ErrCode: code, Message: message}
}

// JSON renders the envelope as a compact JSON string for stdout.
func (e Envelope) JSON() string {
	b, err := json.Marshal(e)
	if err != nil {
		// Marshalling a fixed-shape struct cannot realistically fail; fall back
		// to a hand-built error envelope so callers always get valid JSON.
		return `{"ok":false,"error":"internal_error","message":"failed to encode response"}`
	}
	return string(b)
}
```

// Package cli parses arguments and dispatches to command handlers. It contains a
// deliberately tiny flag parser (no third-party deps, per the stdlib-only rule).
package cli

import "strings"

// ParseFlags turns an argument slice into a map. Rules:
//
//   - "--key value"   -> m["key"] = "value"
//   - "--key" (last, or followed by another --flag) -> m["key"] = "true" (boolean)
//   - first bare token -> m["_"] (the single positional argument, e.g. an ID)
//
// This is sufficient for todoz's flat command surface; it intentionally does not
// support repeated flags or "=".
func ParseFlags(args []string) map[string]string {
	m := make(map[string]string)
	for i := 0; i < len(args); i++ {
		a := args[i]
		if strings.HasPrefix(a, "--") {
			key := strings.TrimPrefix(a, "--")
			if i+1 < len(args) && !strings.HasPrefix(args[i+1], "--") {
				m[key] = args[i+1]
				i++
			} else {
				m[key] = "true"
			}
			continue
		}
		if _, has := m["_"]; !has {
			m["_"] = a
		}
	}
	return m
}

// LoadView is the read model returned by the load command: active lists, active
// tasks (with IsOverdue computed), and the current trash.
type LoadView struct {
	Lists []model.List `json:"lists"`
	Tasks []model.Task `json:"tasks"`
	Trash []model.Task `json:"trash"`
}

// cmdLoad returns the full active view plus trash. Optional --list filters tasks
// (and trash) to a single list. IsOverdue is computed against ctx.Today.
func cmdLoad(ctx Ctx, flags map[string]string) response.Envelope {
	st, err := ctx.Store.Load()
	if err != nil {
		return response.Error("io_error", err.Error())
	}
	filter := flags["list"]

	view := LoadView{Lists: []model.List{}, Tasks: []model.Task{}, Trash: []model.Task{}}

	for _, l := range st.ActiveLists() {
		if filter != "" && l.ID != filter {
			continue
		}
		view.Lists = append(view.Lists, l)
	}
	for _, tk := range st.ActiveTasks() {
		if filter != "" && tk.ListID != filter {
			continue
		}
		tk.IsOverdue = model.ComputeOverdue(tk, ctx.Today)
		view.Tasks = append(view.Tasks, tk)
	}
	for _, tk := range st.TrashTasks() {
		if filter != "" && tk.ListID != filter {
			continue
		}
		view.Trash = append(view.Trash, tk)
	}

	// Stable ordering: by date then id, so output is deterministic for callers.
	sort.Slice(view.Tasks, func(i, j int) bool {
		if view.Tasks[i].Date != view.Tasks[j].Date {
			return view.Tasks[i].Date < view.Tasks[j].Date
		}
		return view.Tasks[i].ID < view.Tasks[j].ID
	})
	sort.Slice(view.Lists, func(i, j int) bool { return view.Lists[i].CreatedAt < view.Lists[j].CreatedAt })

	return response.Success(view)
}

// registry maps command names to their handlers. Adding a command is a single
// line here plus its handler in commands.go.
var registry = map[string]Handler{
	"load":          cmdLoad,
	"add-task":      cmdAddTask,
	"update-task":   cmdUpdateTask,
	"complete-task": cmdCompleteTask,
	"delete-task":   cmdDeleteTask,
	"restore-task":  cmdRestoreTask,
	"add-list":      cmdAddList,
	"update-list":   cmdUpdateList,
	"delete-list":   cmdDeleteList,
	"compact":       cmdCompact,

// Run is the program's core: it parses args, dispatches to a handler, records
// the request to the operation log, and returns the JSON string to print plus
// the process exit code (0 success, 1 failure). It never panics on user input.
func Run(args []string) (string, int) {
	start := clock.Now()

	cfg, err := config.Load()
	if err != nil {
		return response.Error("config_error", err.Error()).JSON(), 1
	}

	if len(args) == 0 {
		return finish(cfg, start, "todoz", response.Error("unknown_command", "no command given"))
	}

	name := args[0]
	handler, ok := registry[name]
	if !ok {
		return finish(cfg, start, "todoz "+name, response.Error("unknown_command", "unknown command: "+name))
	}

	flags := ParseFlags(args[1:])
	ctx := Ctx{
		Store: store.New(cfg),
		Today: clock.Now().Format("2006-01-02"),
		Now:   clock.Format(clock.Now()),
	}
	res := handler(ctx, flags)
	return finish(cfg, start, "todoz "+strings.Join(args, " "), res)
}

// finish records the operation to the oplog and converts the envelope into a
// (stdout, exitCode) pair. Oplog write failures are intentionally ignored: the
// primary operation already happened and must still be reported to the caller.
func finish(cfg config.Config, start interface{ }, request string, res response.Envelope) (string, int) {
	_ = start
	entry := oplog.Entry{
		Timestamp:  clock.Format(clock.Now()),
		Request:    request,
		OK:         res.OK,
		DurationMS: 0,
	}
	if res.OK {
		entry.Data = res.JSON()
	} else {
		entry.Error = res.ErrCode
		entry.Message = res.Message
	}
	_ = oplog.Record(cfg.AppLogPath(), entry)

	code := 0
	if !res.OK {
		code = 1
	}
	return res.JSON(), code
}

		Timestamp:  clock.Format(clock.Now()),
		Request:    request,
		OK:         res.OK,
		DurationMS: time.Since(start).Milliseconds(),
	}
	if res.OK {
		entry.Data = res.JSON()
	} else {
		entry.Error = res.ErrCode
		entry.Message = res.Message
	}
	_ = oplog.Record(cfg.AppLogPath(), entry)

	code := 0
	if !res.OK {
		code = 1
	}
	return res.JSON(), code
}

// Package test holds black-box, process-level tests that exercise todoz the way

package test

import (
	"os/exec"
	"strconv"
	"strings"
	"sync"
	"testing"
)

// Spawn N concurrent add-task processes against one shared home and assert that
// every append survived (no lost writes, no corruption).
func TestConcurrentAppendsNoLoss(t *testing.T) {
	dir := t.TempDir()
	bin := dir + "/todoz"
	if out, err := exec.Command("go", "build", "-o", bin, "../cmd/todoz").CombinedOutput(); err != nil {
		t.Fatalf("build: %v\n%s", err, out)
	}
	home := dir + "/data"
	env := func() []string {
		return append([]string{}, "TODO_LIB_HOME="+home, "TODO_APP_NAME=conc")
	}

	// Seed one list.
	seed := exec.Command(bin, "add-list", "--name", "L")
	seed.Env = env()
	seedOut, err := seed.CombinedOutput()
	if err != nil {
		t.Fatalf("seed: %v\n%s", err, seedOut)
	}
	listID := extractID(t, string(seedOut))

	const N = 50
	var wg sync.WaitGroup
	for i := 0; i < N; i++ {
		wg.Add(1)
		go func(i int) {
			defer wg.Done()
			c := exec.Command(bin, "add-task", "--title", "t"+strconv.Itoa(i), "--date", "2026-06-05", "--list", listID)
			c.Env = env()
			if out, err := c.CombinedOutput(); err != nil {
				t.Errorf("worker %d failed: %v\n%s", i, err, out)
			}
		}(i)
	}
	wg.Wait()

	// Load and count tasks.
	load := exec.Command(bin, "load", "--list", listID)
	load.Env = env()
	out, err := load.CombinedOutput()
	if err != nil {
		t.Fatalf("load: %v\n%s", err, out)
	}
	got := strings.Count(string(out), `"id":"task-`)
	if got != N {
		t.Fatalf("expected %d tasks after concurrent appends, got %d\n%s", N, got, out)
	}
}

// extractID pulls the first list id out of an add-list response.
func extractID(t *testing.T, s string) string {
	t.Helper()
	marker := `"id":"`
	i := strings.Index(s, marker)
	if i < 0 {
		t.Fatalf("no id in %s", s)
	}
	rest := s[i+len(marker):]
	j := strings.Index(rest, `"`)
	return rest[:j]
}
